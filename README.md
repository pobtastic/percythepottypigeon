Percy the Potty Pigeon disassembly
==================================

A disassembly of the [Spectrum](https://en.wikipedia.org/wiki/ZX_Spectrum) game
[Percy the Potty Pigeon](https://en.wikipedia.org/wiki/Potty_Pigeon).

Created using [SkoolKit](https://skoolkit.ca).

Browse the latest release:

* [Percy the Potty Pigeon disassembly](https://pobtastic.github.io/percythepottypigeon/) (hexadecimal; mirror [here](http://skoolkit.arcadegeek.co.uk/percythepottypigeon/))
* [Percy the Potty Pigeon disassembly](https://pobtastic.github.io/percythepottypigeon/dec/) (decimal; mirror [here](http://skoolkit.arcadegeek.co.uk/percythepottypigeon/dec/))

To build the current development version, first either install SkoolKit, or
make a clone of the [SkoolKit repository](https://github.com/skoolkid/skoolkit)
and point the environment variable SKOOLKIT_HOME at it. Then:

    $ utils/mkhtml.py
