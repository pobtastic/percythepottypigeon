24210: Command 1: Draw Repeated Tile
Used by the routine at PopulateRoomBuffer.
Command 1: Draw a repeated tile. The following byte specifies how many times to repeat the current tile.
Input
DE Tile counter
HL The room data pointer
Command01_RepeatedTile 24210 INC HL Increment the room data pointer by one.
24211 LD A,(HL) Fetch the tile byte/ repeat counter.
24212 INC HL Increment the room data pointer by one to move past the command byte.
24213 LD B,A Set the repeat counter in B.
RepeatedTile_Loop 24214 PUSH HL Stash the room data pointer, tile counter and repeat counter on the stack.
24215 PUSH DE
24216 PUSH BC
24217 LD D,1 D=1 (flag: advance position only, don't draw).
24219 CALL DrawRoomTile Call DrawRoomTile.
24222 POP BC Restore the repeat counter, tile counter and room data pointer from the stack.
24223 POP DE
24224 POP HL
24225 DEC DE Decrease the tile counter by one.
24226 LD A,E Jump to PopulateRoomBuffer_Error if the tile counter is zero.
24227 OR D
24228 JR Z,PopulateRoomBuffer_Error
24230 DJNZ RepeatedTile_Loop Decrease the repeat counter by one and loop back to RepeatedTile_Loop until all repeats are processed.
