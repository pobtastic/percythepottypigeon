24283: Command 3: Fill Attribute Buffer With Single Colour
Used by the routine at PopulateRoomBuffer.
Command 3: fill the attribute buffer with a single colour, then parse an overlay map to apply per-cell colour changes. The overlay uses a simple encoding:
  • 18 + count: skip N attribute cells
  • 27 + count + colour: repeat a colour N times
  • 36: end of attribute data
  • Any other value: set a single attribute cell directly
Command03_FillAttributes 24283 INC HL Increment the room data pointer by one.
24284 LD C,(HL) C=*HL.
Fill 704 attribute cells at 55296 with the base colour.
24285 LD IX,55296 IX=55296.
24289 LD DE,704 DE=0704.
PopulateRoomBuffer_FillAttributes_Loop 24292 LD (IX+0),C Write C to *IX+0.
24295 INC IX Increment IX by one.
24297 DEC DE Decrease DE by one.
24298 LD A,E Jump back to PopulateRoomBuffer_FillAttributes_Loop until DE is zero.
24299 OR D
24300 JR NZ,PopulateRoomBuffer_FillAttributes_Loop
Now parse the attribute overlay data to apply per-cell changes on top of the base fill colour.
PopulateRoomBuffer_AttributeOverlay 24302 INC HL Increment HL by one.
24303 LD DE,705 DE=0705.
24306 LD IX,55296 IX=55296.
PopulateRoomBuffer_AttributeOverlay_Loop 24310 LD A,(HL) A=*HL.
24311 CP 18 Jump to PopulateRoomBuffer_AttributeSkip if A is 18 (skip command).
24313 JR Z,PopulateRoomBuffer_AttributeSkip
24315 CP 27 Jump to PopulateRoomBuffer_AttributeRepeat if A is 27 (repeat command).
24317 JR Z,PopulateRoomBuffer_AttributeRepeat
24319 CP 36 Jump to PopulateRoomBuffer_EndAttributes if A is 36 (end of attribute data).
24321 JR Z,PopulateRoomBuffer_EndAttributes
Otherwise, write this byte directly as a single attribute value.
24323 LD C,(HL) C=*HL.
24324 LD (IX+0),C Write C to *IX+0.
24327 DEC DE Decrease DE by one.
24328 INC IX Increment IX by one.
24330 LD A,E Jump back to PopulateRoomBuffer_Error if DE is zero.
24331 OR D
24332 JP Z,PopulateRoomBuffer_Error
24335 INC HL Increment HL by one.
24336 JR PopulateRoomBuffer_AttributeOverlay_Loop Jump to PopulateRoomBuffer_AttributeOverlay_Loop.
Attribute overlay command 18: skip over a number of attribute cells.
The following byte gives the skip count.
PopulateRoomBuffer_AttributeSkip 24338 INC HL Increment HL by one.
24339 LD B,(HL) B=*HL.
