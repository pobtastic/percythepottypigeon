27660: Validate Red Bird Position
Used by the routines at Handler_RedBird, UpdateRedBirdMovement, RedBirdDirectionJumpTable and 29863.
Validates the red bird's proposed position (B=Y, C=X) against the room's flight path boundary data. Scans through the boundary table at *Pointer_RedBirdFlightpathData looking for a region that contains the position.
Each entry is 4 bytes: Y-min, Y-max, X-min, X-max. If a valid region is found, the position is stored and carry is clear. If no valid region is found (entry is 0), carry is set.
Input
B Proposed Y position
C Proposed X position
IX Pointer to red bird sprite data
Output
F Carry clear = valid, carry set = out of bounds
