29690: Handler: Room 10
Handles room 10 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room10 29690 LD A,B Copy the phase counter into A.
29691 CP 3 Return if the phase counter is less than 3.
29693 RET C
29694 LD IX,56024 IX=Sprite06_3Wide_X_Position.
29698 PUSH AF Stash the phase counter on the stack.
29699 CP 4 Call Handler_Helicopter if the phase counter is not equal to 4.
29701 CALL NZ,Handler_Helicopter
29704 POP AF Restore the phase counter from the stack.
29705 LD IX,56028 IX=Sprite07_3Wide_X_Position.
29709 CP 3 Call Handler_Balloon if the phase counter is not equal to 3.
29711 CALL NZ,Handler_Balloon
29714 RET Return.
