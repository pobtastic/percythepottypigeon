29164: Frog 2 State Data
Frog2_StunCountdown 29164 DEFB 0 Stun countdown.
Frog2_SoundCounter 29165 DEFB 0 Ascending sound counter for frog 2. Also used as the horizontal jump offset when positioning the frog sprite (multiplied by 4 and added to the platform X position).
Frog2_Descending_Flag 29166 DEFB 0 Descending sound flag for frog 2. Also determines the frog's facing direction: bit 7 set = facing left, bit 7 clear = facing right.
