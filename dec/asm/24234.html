24234: Command 2: Tile With Attribute
Used by the routine at PopulateRoomBuffer.
Command 2: Draw a tile with an explicit attribute.
The following two bytes specify the repeat count and the tile character to draw.
Input
HL Pointer to the room data
Command02_TileWithAttribute 24234 INC HL Increment the room data pointer by one.
24235 LD A,(HL) B=*HL.
24236 LD B,A
24237 INC HL Increment the room data pointer by one.
24238 LD A,(HL) A=*HL.
24239 INC HL Increment the room data pointer by one.
TileWithAttribute_Loop 24240 PUSH AF Stash the tile character, repeat counter, tile counter and room data pointer on the stack.
24241 PUSH BC
24242 PUSH DE
24243 PUSH HL
24244 LD D,0 D=0 (flag: draw tile to screen buffer).
24246 CALL DrawRoomTile Call DrawRoomTile.
24249 POP HL Restore the room data pointer, tile counter, repeat counter and tile character from the stack.
24250 POP DE
24251 POP BC
24252 POP AF
24253 EX AF,AF' Switch to shadow AF to preserve the tile character.
24254 DEC DE Decrease DE by one.
24255 LD A,E Jump back to PopulateRoomBuffer_Error if DE is zero.
24256 OR D
24257 JR Z,PopulateRoomBuffer_Error
24259 EX AF,AF' Switch back to main AF to retrieve the tile character.
24260 DJNZ TileWithAttribute_Loop Decrease the repeat counter by one and loop back to TileWithAttribute_Loop until all tiles are drawn.
24262 JR PopulateRoomBuffer_ParseByte Jump to PopulateRoomBuffer_ParseByte to continue parsing.
