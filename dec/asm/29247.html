29247: Handler: Room 1
Handles room 1 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room01 29247 LD A,B A=B.
29248 CP 1 Return if the phase counter is 1.
29250 RET Z
Set up the first object.
29251 LD IX,56024 IX=Sprite06_3Wide_X_Position.
29255 LD IY,29960 IY=29960.
29259 PUSH AF Stash the phase counter on the stack.
29260 CALL 29753 Call 29753.
29263 POP AF Restore the phase counter from the stack.
Set up the second object; only processed during phase 3.
29264 LD IX,56028 IX=Sprite07_3Wide_X_Position.
29268 LD IY,29965 IY=29965.
29272 CP 3 Is the phase counter equal to 3?
29274 PUSH AF Stash the phase counter on the stack.
29275 CALL Z,29757 Call 29757 if the phase counter is 3.
29278 POP AF Restore the phase counter from the stack.
From phase 4 onwards, call UpdatePlaneFlightPath_12.
29279 CP 4 Call UpdatePlaneFlightPath_12 if the phase counter is 4 or greater.
29281 CALL NC,UpdatePlaneFlightPath_12
29284 RET Return.
