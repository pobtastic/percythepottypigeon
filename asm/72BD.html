72BD: Handler: Room 04
Handles room 04 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room04 72BD LD A,B Copy the phase counter into A.
72BE CP $01 Return if A is equal to 01.
72C0 RET Z
72C1 PUSH AF Stash the phase counter on the stack.
72C2 LD IX,$DAD0 IX=Sprite04_3Wide_X_Position.
72C6 CP $05 Call Handler_Helicopter if the phase counter is not equal to 05.
72C8 CALL NZ,Handler_Helicopter
72CB POP AF Restore the phase counter from the stack.
72CC LD IX,$DAD4 IX=Sprite05_3Wide_X_Position.
72D0 PUSH AF Stash the phase counter on the stack.
72D1 CP $03 Call Handler_Plane if the phase counter is greater than 03.
72D3 CALL NC,Handler_Plane
72D6 POP AF Restore the phase counter from the stack.
72D7 LD IX,$DAD8 IX=Sprite06_3Wide_X_Position.
72DB CP $04 Call Handler_Balloon if the phase counter is equal to 04.
72DD CALL Z,Handler_Balloon
72E0 RET Return.
