73E1: Handler: Room 09
Handles room 09 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room09 73E1 LD A,B Copy the phase counter into A.
73E2 CP $01 Return if the phase counter is equal to 01.
73E4 RET Z
73E5 LD IX,$DAD0 IX=Sprite04_3Wide_X_Position.
73E9 PUSH AF Stash the phase counter on the stack.
73EA CP $03 Call Handle_Red_Bird if the phase counter is not equal to 03.
73EC CALL NZ,Handle_Red_Bird
73EF POP AF Restore the phase counter from the stack.
73F0 LD IX,$DAD8 IX=Sprite06_3Wide_X_Position.
73F4 CP $03 Call Handle_Parachute_Descent if the phase counter is greater than 03.
73F6 CALL NC,Handle_Parachute_Descent
73F9 RET Return.
