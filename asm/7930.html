7930: Generate Random Number
Used by the routines at Handle_Red_Bird, Randomise_Position_Loop, Animate_Stunned_RedBird, 7512 and UpdatePlaneFlightPath.
Generates a pseudo-random number using a linear decay formula. Reads the current seed from 7AC7, computes (seed + 01) * FE then derives the new seed from the difference of the low and high bytes of the result. Returns the new value in A.
