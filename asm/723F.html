723F: Handler: Room 01
Handles room 01 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room01 723F LD A,B A=B.
7240 CP $01 Return if the phase counter is 01.
7242 RET Z
Set up the first object.
7243 LD IX,$DAD8 IX=DAD8.
7247 LD IY,$7508 IY=7508.
724B PUSH AF Stash the phase counter on the stack.
724C CALL $7439 Call 7439.
724F POP AF Restore the phase counter from the stack.
Set up the second object; only processed during phase 03.
7250 LD IX,$DADC IX=DADC.
7254 LD IY,$750D IY=750D.
7258 CP $03 Is the phase counter equal to 03?
725A PUSH AF Stash the phase counter on the stack.
725B CALL Z,$743D Call 743D if the phase counter is 03.
725E POP AF Restore the phase counter from the stack.
From phase 04 onwards, call 77B9.
725F CP $04 Call 77B9 if the phase counter is 04 or greater.
7261 CALL NC,$77B9
7264 RET Return.
