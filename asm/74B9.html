74B9: Animate Stunned Red Bird
Used by the routines at Handle_Red_Bird, Handle_Cat, UpdatePlaneFlightPath, Handle_Walking_Paratrooper and Handle_Parachute_Descent.
Animates the stunned red bird sequence. Increments the animation counter until it reaches 85 cycling through animation frames. Assigns a random colour attribute and plays a hit sound each frame.
Animate_Stunned_RedBird 74B9 LD A,(IY+$00) Return if the animation counter at *IY+00 has reached 85 (animation complete).
74BC CP $85
74BE RET Z
74BF INC A Increment the animation counter and write it back to *IY+00.
74C0 LD (IY+$00),A
74C3 AND %01111111 Mask off bit 7, rotate right and add 36 to calculate the animation frame ID.
74C5 RRCA
74C6 ADD A,$36
74C8 LD (IX+$03),A Write the frame ID to *IX+03.
74CB CALL Generate_Random_Number Generate a random number and mask with 07 to produce a random colour attribute.
74CE AND %00000111
74D0 LD (IX+$02),A Write the colour attribute to *IX+02.
74D3 JP PlayHitSound Jump to PlayHitSound.
