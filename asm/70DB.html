70DB: Update Frog
Used by the routine at Handler_Frogs.
Updates a single frog's behaviour through the jump cycle.
Input
BC Ground position
HL Pointer to frog state data
IX Pointer to frog sprite data
UpdateFrog 70DB LD A,(HL) Fetch the timer value for the frog.
70DC OR A Jump to UpdateFrog_SetIdle if the timer is zero (the frog is idle).
70DD JR Z,UpdateFrog_SetIdle
70DF DEC (HL) Decrement the timer by one.
70E0 JR Z,UpdateFrog_SetIdle Jump to UpdateFrog_SetIdle if the timer has now reached zero.
This Frog is mid-jump, cycle through the animation frames.
