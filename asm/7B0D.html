7B0D: Paratrooper Room State Data
Per-room state data for the walking paratrooper, with four bytes per room. Indexed by (room number - 01) * 04. Only rooms 05 06 and 08 have active entries.
Offset Description
00 Stunned flag (bit 7 set = stunned by egg)
01 Platform Y position
02 Left boundary X position
03 Right boundary X position
