730E: Handler: Room 06
Handles room 06 logic (the starting screen). The phase counter in B controls which objects are initialised or activated on each pass, progressively setting up more objects as the phase increases.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room06 730E LD A,B A=B.
First object; initialised during phases 01 03 and 04.
730F LD IX,$DAC8 IX=Spider_X_Position.
7313 PUSH AF Stash the phase counter on the stack.
7314 CP $01 Call Handler_Spider if the phase counter is 01, 03 or 04.
7316 CALL Z,Handler_Spider
7319 CP $03
731B CALL Z,Handler_Spider
731E CP $04
7320 CALL Z,Handler_Spider
7323 POP AF Restore the phase counter from the stack.
Second object; return early if still in phase 01.
7324 LD IX,$DACC IX=DACC.
7328 CP $01 Return if the phase counter is 01.
732A RET Z
732B PUSH AF Stash the phase counter on the stack.
732C CALL $7512 Call 7512.
732F POP AF Restore the phase counter from the stack.
Third object; initialised during phase 02 then handled every phase after.
7330 LD IX,$DAD0 IX=DAD0.
7334 CP $02 Is the phase counter 02?
7336 PUSH AF Stash the phase counter on the stack.
7337 CALL Z,$759A Call 759A if the phase counter is 02.
733A POP AF Restore the phase counter from the stack.
733B RET Z Return if the phase counter was 02.
733C PUSH AF Stash the phase counter on the stack.
733D CALL $7439 Call 7439.
7340 POP AF Restore the phase counter from the stack.
7341 CP $03 Return if the phase counter is 03.
7343 RET Z Return.
Fourth object; initialised during phase 04.
7344 LD IX,$DAD4 IX=DAD4.
7348 PUSH AF Stash the phase counter on the stack.
7349 LD IY,$750D IY=750D.
734D CP $04 Call 743D if the phase counter is 04.
734F CALL Z,$743D
7352 POP AF Restore the phase counter from the stack.
7353 CP $04 Return if the phase counter is 04.
7355 RET Z Return.
Fifth object; active from phase 05 onwards.
7356 LD IX,$DAD8 IX=DAD8.
735A PUSH AF Stash the phase counter on the stack.
735B CALL $7893 Call 7893.
735E POP AF Restore the phase counter from the stack.
Update the fourth object.
735F LD IX,$DAD4 IX=DAD4.
7363 CALL $7680 Call 7680.
7366 RET Return.
