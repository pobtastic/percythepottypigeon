73FA: Handler: Room 0A
Handles room 0A logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room10 73FA LD A,B Copy the phase counter into A.
73FB CP $03 Return if the phase counter is less than 03.
73FD RET C
73FE LD IX,$DAD8 IX=Sprite06_3Wide_X_Position.
7402 PUSH AF Stash the phase counter on the stack.
7403 CP $04 Call Handler_Helicopter if the phase counter is not equal to 04.
7405 CALL NZ,Handler_Helicopter
7408 POP AF Restore the phase counter from the stack.
7409 LD IX,$DADC IX=Sprite07_3Wide_X_Position.
740D CP $03 Call Handler_Balloon if the phase counter is not equal to 03.
740F CALL NZ,Handler_Balloon
7412 RET Return.
