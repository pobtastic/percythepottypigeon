73A4: Handler: Room 08
Handles room 08 logic. The phase counter in B controls which objects are initialised or updated on each pass.
Input
B Room phase counter (from *CurrentLevel)
Handler_Room08 73A4 LD A,B Copy the phase counter into A.
73A5 LD IX,$DAC8 IX=Sprite02_3Wide_X_Position.
73A9 PUSH AF Stash the phase counter on the stack.
73AA CALL Handler_Helicopter Call Handler_Helicopter.
73AD POP AF Restore the phase counter from the stack.
73AE LD IX,$DACC IX=Sprite03_3Wide_X_Position.
73B2 CP $01 Return if the phase counter is equal to 01.
73B4 RET Z
73B5 PUSH AF Stash the phase counter on the stack.
73B6 CP $03 Call Handler_Cat if the phase counter is not equal to 03.
73B8 CALL NZ,Handler_Cat
73BB POP AF Restore the phase counter from the stack.
73BC CP $02 Return if the phase counter is equal to either 02 or 03.
73BE RET Z
73BF CP $03
73C1 RET Z
73C2 PUSH AF Stash the phase counter on the stack.
73C3 LD IX,$DAD0 IX=Sprite04_3Wide_X_Position.
73C7 CALL Handler_Balloon Call Handler_Balloon.
73CA POP AF Restore the phase counter from the stack.
73CB PUSH AF Stash the phase counter on the stack.
73CC LD IX,$DAD8 IX=Sprite06_3Wide_X_Position.
73D0 CP $03 Call Handler_Parachute_Descent if the phase counter is greater than 03.
73D2 CALL NC,Handler_Parachute_Descent
73D5 POP AF Restore the phase counter from the stack.
73D6 CP $04 Return if the phase counter is equal to 04.
73D8 RET Z
73D9 LD IX,$DAD4 IX=Sprite05_3Wide_X_Position.
73DD CALL Handler_Walking_Paratrooper Call Handler_Walking_Paratrooper.
